evals.intentsolutions.io — the predicate authority

This host is the canonical home of the Intent Eval Platform's predicate URIs — the identifiers inside every signed attestation the platform publishes. When a signed statement says its predicate is https://evals.intentsolutions.io/gate-result/v1, this is the page that URI identifies: what the statement asserts, where its machine-readable schema lives, and how to verify it.

Looking for results? They live at labs.intentsolutions.io — graded nightly, signed, and re-verified before rendering. This host defines; that host shows.

Integrity properties of this host

DNSSEC-signed zone; CAA-pinned to Let's Encrypt; predicate URIs never move hosts (ISEDC CISO binding). A predicate page may gain clarifications; the meaning a URI identifies is immutable — breaking changes mint a new version segment, never rewrite an old one.

Predicate URIs are identifiers bound to this host by the contracts kernel (@intentsolutions/core) under an ISEDC council binding: they live here and only here — never at labs.*. This host serves the human-readable definition each URI identifies; the machine-readable schemas live version-pinned in the kernel. Results live at labs.intentsolutions.io. Best-effort, single-operator service.
